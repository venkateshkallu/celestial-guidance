# Jyotish Guru - Astrology Services Website

A professional astrology services website featuring Kundli, marriage, love, career, health consultations and more.

## Features

- Responsive design with sticky header
- Services section with various astrology offerings
- About section with expert astrologer profile
- Testimonials from clients
- Contact section with WhatsApp integration
- Trust badges and certifications

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev
```

## Build

```bash
bun run build
```
